# Attendance System

This repository contains the complete source code for the Attendance System. The project is structured into two main components to separate the backend/web environment from the mobile environment.

## Project Structure

* **`laravel/` (Web & API Backend)**: Built with the Laravel framework. This handles the database management (Supabase), the web-based interface, and serves as the REST API for the mobile application.
* **`flutter/` (Android Mobile App)**: Built with the Flutter framework. This is a dedicated Android application designed specifically for employees to submit their daily attendance.

---

## 💻 Tech Stack
- **Mobile**: Flutter SDK (>=2.18.0 <4.0.0)
- **Web/Backend**: Laravel, PHP
- **Database**: Supabase (PostgreSQL)
